# Glossário

## Termos principais
- **Portfólio**: conjunto de ativos pertencentes a um usuário, com valor total definido.
- **Classe de ativo**: agrupamento lógico (ex.: Stocks, Crypto, Commodities).
- **Ativo**: item individual (ex.: AAPL, BTC).
- **Meta (classe)**: % desejado de uma classe dentro do portfólio.
- **Meta (ativo)**: % desejado de um ativo dentro da classe.
- **Cotação**: preço unitário atual do ativo.
- **Valor**: quantidade × cotação.
- **Desvio**: diferença entre % atual e % meta.
- **Rebalanceamento**: ajuste sugerido para voltar à meta.
- **OCR**: extração automática de texto em imagens (Tesseract).
- **Template**: variação de layout do dashboard.
